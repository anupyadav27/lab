# Security Awareness Training Workflow

## Overview

This workflow guides security teams through developing and deploying comprehensive security awareness training programs. It covers human risk assessment, content design, phishing simulations, delivery strategy, and metrics-driven continuous improvement.

## Workflow Steps

| Step | File | Description |
|------|------|-------------|
| 1 | step-01-init.md | Initialize workflow, assess current program |
| 1b | step-01b-continue.md | Resume workflow from previous session |
| 2 | step-02-risk-assessment.md | Identify threats and high-risk populations |
| 3 | step-03-content-development.md | Design training curriculum |
| 4 | step-04-phishing-simulation.md | Plan phishing campaign strategy |
| 5 | step-05-delivery-strategy.md | Plan rollout and LMS integration |
| 6 | step-06-metrics-measurement.md | Define KPIs and reporting |
| 7 | step-07-continuous-improvement.md | Plan ongoing program evolution |

## Primary Agents

- **Sentinel** (Compliance Guardian) - Compliance requirements and governance
- **Shield** (Blue Team Lead) - Program strategy and metrics
- **Phoenix** (Incident Commander) - Response to awareness failures

## Frameworks Referenced

- NIST CSF (Cybersecurity Framework)
- CIS Controls v8
- SANS Security Awareness Maturity Model

## Output

The workflow produces a comprehensive security awareness training program document including:
- Program overview and current state assessment
- Human risk assessment with population prioritization
- Training content design with role-based tracks
- Phishing simulation strategy
- Delivery and rollout plan
- Metrics and measurement framework
- Continuous improvement roadmap

## Templates

- `training-plan-template.md` - Annual training plan template
- `phishing-campaign-template.md` - Phishing campaign planning template

## Usage

Invoke via agent menu:
- Shield: [SA] Security Awareness Training Workflow
- Sentinel: Through chat or action menu

Or directly:
```
Load and execute: workflows/security-awareness-training/workflow.md
```
